What This App Does

📋

Content Scheduling

Pre-written, pre-approved educational content is queued by the account owner and posted on a set schedule via the TikTok Content Posting API.

✅

Human Approval Gate

Every post requires explicit sign-off from Jaydon Marley before it goes live. Nothing is ever published automatically without review.

🔒

No Data Collection

This app does not collect, store, or share any user data. It posts content outbound only — no follower data, comments, or analytics are accessed.

How It Works

1

Content is created and reviewed

Jaydon writes and approves educational Nutrilite supplement content including captions, IBO disclosures, and video assets.

2

Video is uploaded via API

The approved video file is uploaded to TikTok's servers using the video.upload scope of the Content Posting API.

3

Post is published to @marleys.wellness

The uploaded video is published to the TikTok account using video.publish, with the pre-written caption and required IBO disclosure.

4

Nothing else happens

No DMs, no scraping, no follower data, no engagement automation. This app does one thing: post approved content.
